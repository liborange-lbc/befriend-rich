## 新增需求

### 需求:今日概览
大盘看板页面必须展示今日资产总览信息，包括总资产（人民币）、日涨跌幅/金额、当日汇率。

#### 场景:展示今日概览
- **当** 用户访问大盘看板页面
- **那么** 页面顶部展示总资产、日涨跌幅、日涨跌金额、USD/CNY 汇率

### 需求:基金行情列表
看板必须展示所有启用基金的当日行情，包括名称、最新净值、日涨跌幅，支持点击选中查看详情。

#### 场景:展示基金行情
- **当** 用户查看看板
- **那么** 页面展示基金列表，每只基金显示名称、净值、日涨跌幅和涨跌方向标识

#### 场景:点击基金查看K线
- **当** 用户点击列表中的某只基金
- **那么** 右侧展示该基金的 K 线图（收盘价 + MA 均线）

### 需求:K线图与时间控制
看板必须展示选中基金的收盘价走势和均线，时间范围通过滑块控制。

#### 场景:展示K线图
- **当** 用户选中一只基金
- **那么** 图表展示收盘价折线和 MA30/MA60/MA120 均线，支持缩放

#### 场景:时间范围滑块
- **当** 用户拖动时间滑块
- **那么** 图表实时更新显示范围，无需重新加载数据

### 需求:均值偏差热力图
看板必须展示所有启用基金的均值偏差热力图，用颜色深浅直观表达高估/低估程度。

#### 场景:展示热力图
- **当** 用户查看看板
- **那么** 页面展示矩阵式热力图：行为基金，列为 MA30/60/90/120/180/360，颜色表示偏差方向和程度（绿色=低估，红色=高估）

### 需求:策略提醒展示
看板必须展示最近的策略提醒记录。

#### 场景:展示策略提醒
- **当** 用户查看看板
- **那么** 页面底部展示最近的策略提醒列表（时间、基金、触发条件、当前值）
